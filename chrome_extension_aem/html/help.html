How to use this extension?
Log in to the selected version of AEM. Enter page url and use buttons that open the page in the selected environment. Or use shortcuts that open various AEM tools. Or use buttons modifying the active page.

In a standard installation AEM creates a new version of a page or node when you activate a page after updating the content. The Purge Versions tool is intended for purging the versions of a node or a hierarchy of nodes in your repository. Its primary purpose is to help you reduce the size of your repository by removing old versions of your nodes. Version Purging
